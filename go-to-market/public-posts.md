# Public Post Drafts

## Short Developer Post

I have been exploring a practical model for agentic coding.

The biggest opportunity may not be "AI writes code." It may be what happens before and around the code:

- validate Jira requirements
- score build readiness
- ask Product or developers targeted questions
- inspect LWC, Apex, metadata, and permissions
- attempt sandbox builds only when confidence is high enough
- produce developer ready handoffs
- require human authorization before promotion

I call this confidence gated engineering.

Agents can accelerate delivery, but humans still own judgment, review, and release authority.

## Salesforce Implementation Pack Post

For Salesforce teams, agentic engineering has to be more than code generation.

An LWC change can involve Apex controllers, object fields, permission sets, FLS, Lightning page targets, Experience Cloud behavior, Jest tests, Apex tests, and release region promotion.

That is why I am interested in confidence gated delivery: using agents to score Jira requirements, surface ambiguity early, attempt sandbox validation when ready, and hand developers evidence before work moves through Copado, AutoRABIT, or another release pipeline.

The goal is faster delivery with better control.

## Leadership Post

The most useful AI engineering workflows will not remove human accountability. They will move human judgment to better points in the delivery process.

I have been working on a confidence gated model for agentic coding:

- requirements are scored before development starts
- ambiguity becomes targeted Product or developer questions
- sandbox builds happen only above defined confidence thresholds
- validation evidence is attached before review
- developers authorize promotion to the next region

This is the operating model I think regulated enterprise teams should be watching: agents accelerate the path from requirement to evidence, while humans retain authority over judgment and release.

## Vendor Neutral Comment

What I am most interested in is not which assistant writes the first draft of code. It is how tools like Microsoft Copilot, OpenAI Codex, Anthropic Claude Code, Salesforce, Copado, and AutoRABIT could fit into a governed delivery system:

requirements readiness, sandbox validation, developer review, and authorized promotion.
