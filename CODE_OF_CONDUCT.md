# Code Of Conduct

This project is intended to be a useful, respectful space for people discussing agentic coding, requirements readiness, release management, and confidence gated engineering workflows.

Please:

- be direct, constructive, and specific
- critique ideas and tools, not people
- avoid confidential or proprietary examples
- respect that teams have different release constraints and risk profiles

Unwelcome behavior includes harassment, personal attacks, or sharing private information without permission.
