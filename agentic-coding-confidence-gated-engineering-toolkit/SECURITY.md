# Security

This toolkit is a public draft and should not be used with confidential data unless you have reviewed and adapted it for your environment.

Do not include secrets, access tokens, internal Jira issues, Salesforce org data, customer data, or proprietary implementation details in public issues or examples.

## Reporting A Concern

If you find a security concern in the starter utility or public templates, open a GitHub issue with a sanitized description.

Do not include live credentials, internal URLs, or nonpublic data.
